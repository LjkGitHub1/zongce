# 综测材料收集系统 - 前端

## 技术栈
- Vue 3
- Element Plus
- Plus Pro Components
- Pinia
- Vue Router
- Axios
- Vite

## 安装步骤

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 构建生产版本
```bash
npm run build
```

## 项目结构
```
frontend/
├── src/
│   ├── api/          # API接口
│   ├── assets/       # 静态资源
│   ├── components/   # 组件
│   ├── layouts/      # 布局
│   ├── router/       # 路由
│   ├── stores/       # 状态管理
│   ├── utils/        # 工具函数
│   ├── views/        # 页面
│   └── main.js       # 入口文件
└── package.json
```
